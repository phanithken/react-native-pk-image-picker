# react-native-pk-image-picker

A customizable react-native image picker

## Installation

```sh
npm install react-native-pk-image-picker
```

## Usage

```js
import { PkImagePickerView } from "react-native-pk-image-picker";

// ...

<PkImagePickerView color="tomato" />
```
